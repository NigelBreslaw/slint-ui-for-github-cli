# Primer — which globals to import

Quick reference when wiring a component or view. Package rules and verification: [`AGENTS.md`](../AGENTS.md); token layers and exports: [`readme.md`](../readme.md#design-tokens) and [`primer.slint`](primer.slint).

- **App views / chrome:** `PrimerColors` and `LayoutTokens` (from `tokens.slint` or [`primer.slint`](primer.slint) barrel).
- **Controls** (GitHub button palette, danger/success): `ButtonTokens` + `PrimerColors` as needed — see [`Buttons/buttons.slint`](Buttons/buttons.slint), [`Select/select.slint`](Select/select.slint).
- **CounterLabel** (pill): [`CounterLabel/counter-label.slint`](CounterLabel/counter-label.slint) — pass explicit colors from `Button`, or set `use-primer-scheme` with `CounterLabelVariant` for standalone `PrimerColors` (`bgColor-neutral-emphasis` / `neutral-muted`, `counter-borderColor`).
- **Label** (product metadata chip): `LabelTokens` + `LayoutTokens` — [`Label/label.slint`](Label/label.slint), [`Label/logic.slint`](Label/logic.slint); not the same as **CounterLabel**.
- **LabelGroup:** `Label` + `LayoutTokens` only — [`LabelGroup/label-group.slint`](LabelGroup/label-group.slint); no separate color global.
- **DataTable:** [`DataTable/data-table.slint`](DataTable/data-table.slint) — `LayoutTokens` + `PrimerColors`; `Label` + `LabelVariant` / `LabelSize` from [`Label/types.slint`](Label/types.slint); `Image` for `iconText`; `IconButton` (`IconButtonVariant.invisible`, `Size.small`) for `action`. Models: [`DataTable/types.slint`](DataTable/types.slint). `Icons.*` from [`assets/icons.slint`](assets/icons.slint); sort: `Icons.sort_asc` / `Icons.sort_desc`. No separate DataTable color global.
- **TableContainer:** [`DataTable/table-container.slint`](DataTable/table-container.slint) — `Button` + `ButtonVariant` + `LayoutTokens` + `PrimerColors` + `Size`; content via `@children`.
- **TreeView:** [`TreeView/primer-tree-view.slint`](TreeView/primer-tree-view.slint) — **`[TreeViewRow] rows`**, optional **`max-gutter-columns`** (indent guide slots, default **12**), **`TreeViewRow`** / **`TreeViewTrailingVisual`** in [`TreeView/types.slint`](TreeView/types.slint); **`TreeViewTokens`** + **`LayoutTokens`** + **`Icons`**; root **`flat`** / **`truncate`**; **`row-toggle-requested(string)`**; **`row-secondary-actions-requested(string, length, length, length, length)`** when rows use **`has-secondary-actions`** (anchor args for **`AnchoredOverlay`**). API notes: [`TreeView/API.md`](TreeView/API.md).
- **Banner:** `BannerTokens` + `LayoutTokens` + `PrimerColors` for default text fg — [`Banner/banner.slint`](Banner/banner.slint).
- **Avatar:** `AvatarTokens` + `LayoutTokens` — [`Avatar/avatar.slint`](Avatar/avatar.slint); optional **`alt-text`** (maps to **`Image`** accessibility); **`@children`** for overlays (**`TouchArea`** in **`AppHeader`**).
- **Checkbox:** `CheckboxTokens` + `LayoutTokens` + `PrimerColors` — [`Checkbox/checkbox.slint`](Checkbox/checkbox.slint); icons: `Icons` (`checkbox`, `checkbox_fill`).
- **ToggleSwitch:** `ToggleSwitchTokens` + `LayoutTokens` + `PrimerColors` + `Size` + `Icons` + [`Spinner`](Progress/spinner.slint) (embedded when **`loading`**) — [`ToggleSwitch/toggle-switch.slint`](ToggleSwitch/toggle-switch.slint); status vs track **`TouchArea`** split; **`FocusScope`** on the track; optional **`reduce-motion`** (**0ms** transitions, upstream **`prefers-reduced-motion`**).
- **CheckboxGroup:** `LayoutTokens` + `PrimerColors` + `ButtonTokens` (validation line) — [`CheckboxGroup/checkbox-group.slint`](CheckboxGroup/checkbox-group.slint); reuse `ValidationStatus` from [`Select/select.slint`](Select/select.slint). When the group is `disabled`, mirror `disabled` on each child `Checkbox`.
- **Radio:** `RadioTokens` + `LayoutTokens` + `PrimerColors` — [`Radio/radio.slint`](Radio/radio.slint); mutex **`value`** + parent-owned **`checked`** (see [`RadioGroup`](RadioGroup/radio-group.slint) gallery).
- **RadioGroup:** same globals as **CheckboxGroup** — [`RadioGroup/radio-group.slint`](RadioGroup/radio-group.slint); reuse `ValidationStatus` from [`Select/select.slint`](Select/select.slint). When the group is `disabled`, mirror `disabled` on each child `Radio`.
- **PrimerTextInput:** `TextInputTokens` + `LayoutTokens` + `PrimerColors` + `ButtonTokens` (disabled fg, counter danger) — [`PrimerTextInput/primer-text-input.slint`](PrimerTextInput/primer-text-input.slint); **`TextInputLoaderPosition`** + **`Spinner`** when `loading`; **`character-limit`** (`<= 0` off) + **`character-limit-exceeded-message`** (validation row mirrors `validation-message`, driven by **`text`** vs limit); **`label-visually-hidden`**; **`has-trailing-action`** ([`IconButton`](Buttons/buttons.slint) clear, `IconButtonVariant.invisible`, `Size.small`); reuse `ValidationStatus` from [`Select/select.slint`](Select/select.slint); leading/trailing slots use `Icons.*` from [`assets/icons.slint`](assets/icons.slint).
- **AnchoredOverlay** (anchored `PopupWindow` shell): [`AnchoredOverlay/anchored-overlay.slint`](AnchoredOverlay/anchored-overlay.slint) — **`AnchoredOverlaySide`** / **`AnchoredOverlayAlign`** in [`AnchoredOverlay/types.slint`](AnchoredOverlay/types.slint); viewport clamp / flip uses **`AppWindow`** from [`tokens.slint`](tokens.slint); optional **`slot-padding`**, **`panel-border-radius`**, **`panel-border-width`**, **`panel-border-color`**, **`panel-fill`**, **`panel-elevation-shadow`** (defaults **`OverlayTokens`** — unchanged visuals when unset); optional **`panel-width`**, **`anchor-*`** from trigger; **`shrink-wrap`** compact menus — not modal **`backdrop-scrim`**; see [`anchored-popupwindow.md`](../../slint-gallery/ui/views/anchored-popupwindow.md).
- **ModalOverlay** (viewport modal `PopupWindow` + scrim + panel): [`AnchoredOverlay/modal-overlay.slint`](AnchoredOverlay/modal-overlay.slint) — **`OverlayTokens.backdrop-scrim`** + **`OverlayPanelChrome`** with **`OverlayTokens`** panel tokens; **center**: backdrop opacity + panel **`Overlay--motion-scaleFade`** (**`DialogTokens.overlay-enter-duration`**); **side sheet** (**`narrow-viewport`** **false**, **`panel-position`** **left** / **right**): horizontal slide (**`DialogTokens.overlay-sheet-enter-duration`**), full viewport height, asymmetric corners; **`narrow-viewport`** **true** + **`narrow-position`**: **`bottom`** — full-width **`slideUp`** (**`overlay-sheet-enter-duration`**), top-rounded / bottom-flush chrome; **`fullscreen`** — full window **`scaleFade`** + **`0px`** radii; **`center`** — same **`scaleFade`** path as regular center; viewport clamp uses **`OverlayTokens.modal-viewport-max-*`** on **`panel-width`** / **`panel-max-height`**; **`panel-vertical-align`** (**`DialogAlignPreset`**, ignored for side sheets and narrow bottom / fullscreen); **`slot-padding`** (default **`0px`** for dialog shell).
- **OverlayPanelChrome** (reusable panel fill + shadow + border + padded slot): [`AnchoredOverlay/overlay-panel-chrome.slint`](AnchoredOverlay/overlay-panel-chrome.slint) — used by **`AnchoredOverlay`** and **`ModalOverlay`**; pass **`elevation-border-radius`** (shadow) plus per-corner **`surface-border-*-radius`** (fill/border clip — side sheets **`0px`** on the flush edge), surface colors / shadow (**`OverlayTokens`** `out` values), **`slot-padding`** / **`slot-fill-height`**.
- **DialogTokens** (`tokens.slint`) — Primer **Dialog** spacing, typography, **`width-*`** (**React **`widthMap`** literals where tokens differ), **`height-small`** / **`height-large`** (**React **`heightMap`**), motion durations (**`overlay-enter-duration`**, **`overlay-sheet-enter-duration`**); composes **`LayoutTokens`** `out`; audit table for **`overlay.size`** parity — no stray color literals.
- **Dialog** (public modal shell — full-panel **`@children`**) [`Dialog/dialog.slint`](Dialog/dialog.slint) — viewport modal via **`ModalOverlay`** (clipped inner slot); **`width-preset`**, **`height-preset`**, **`align-preset`**, **`position-preset`**, **`narrow-position-preset`**, **`fill-mode`**, and **`narrow-layout`** mirror upstream Primer **Dialog** layout rules. Default Storybook-style chrome is not a second component: stack **`StandardDialogHeader`**, **`DialogBody`**, and **`DialogFooter`** as children. **[`types.slint`](Dialog/types.slint):** **`DialogWidthPreset`**, **`DialogHeightPreset`**, **`DialogAlignPreset`**, **`DialogPositionPreset`**, **`DialogNarrowPositionPreset`**, **`DialogFillMode`**, **`DialogFooterButtonLayout`** (PR10), **`DialogFooterAutoFocus`** (PR11). Parts: [`Dialog/parts/dialog-header.slint`](Dialog/parts/dialog-header.slint), [`Dialog/parts/dialog-body.slint`](Dialog/parts/dialog-body.slint), [`Dialog/parts/dialog-footer.slint`](Dialog/parts/dialog-footer.slint) — **`DialogFooter`** **`use-default-actions`** (**`true`** = default three-**`Button`** row; **`false`** + **`HorizontalLayout`** **`@children`** for custom buttons; use **`visible:`** branches, not **`if`**, because **`@children`** cannot live inside **`if`**). **Storybook vs gallery:** see **Dialog** in [`readme.md`](readme.md); **`pnpm dev:gallery`** → **Dialogs**.
- **ActionList** (vertical list — Storybook-aligned shell + row): [`ActionList/action-list.slint`](ActionList/action-list.slint) — **`[ActionListLine] lines`**, **`ActionListLineKind`**, **`ActionListSelectionMode`**, **`show-dividers`**, **`item-activated`**; models in [`ActionList/types.slint`](ActionList/types.slint). Rows: **`ActionListRow`** ([`ActionList/action-list-row.slint`](ActionList/action-list-row.slint)) — **`ButtonTokens`** + **`ActionListTokens`** + **`CheckboxTokens`** for selection chrome; optional **`ActionListItemDivider`** ([`ActionList/action-list-item-divider.slint`](ActionList/action-list-item-divider.slint)) for standalone item dividers. **`ActionListTokens`** + **`LayoutTokens`** (**`action-list-item-min-height`**, **`action-list-avatar-diameter`**, divider tokens).
- **SelectPanel** (panel body — title, filter, list, optional footer): [`SelectPanel/select-panel.slint`](SelectPanel/select-panel.slint) — **`SelectPanelMode`**, **`SelectPanelItem`** in [`SelectPanel/types.slint`](SelectPanel/types.slint); compose with **`AnchoredOverlay`**; **`show-footer`** + **`@children`** for Primer-style footer actions; **`loading-message`**, **`empty-title`** / **`empty-message`**, **`focused-index`**; single mode uses **`SelectOption`** + **`ActionListRow`**; multi mode uses **`SelectPanelRow`** + **`Checkbox`** (`interactive: false` in row) + **`multi-selected`** (`[bool]`, out-of-range unchecked) + **`multi-item-activated`** (parent or a gallery **`global`** + Node bridge updates the slice); **`Icons.search`** on **`PrimerTextInput`**; **`Spinner`** + **`Size`** when `loading`; `LayoutTokens` + `PrimerColors` + `ButtonTokens` / `CheckboxTokens`.
- **FilteredActionList** (filter + scrollable **`ActionList`**): [`FilteredActionList/filtered-action-list.slint`](FilteredActionList/filtered-action-list.slint) — **`filter-text`** / **`filter-changed`**, **`placeholder`**, **`lines`** (`[ActionListLine]`), **`show-dividers`**, **`selection-mode`** / **`selected-index`** / **`multi-selected`**, **`item-activated`**, **`body-region-height`** (default **`SelectPanelTokens.list-max-height-default`**), **`loading`** / **`loading-kind`** ([`FilteredActionList/types.slint`](FilteredActionList/types.slint) **`FilteredActionListLoadingKind`**: **`input`**, **`body_spinner`**, **`body_skeleton`**), **`loading-message`** (body spinner); **`show-message`**, **`message-title`**, **`message-description`** (body replaces list after loading); **`empty-title`**, **`empty-message`** when **`lines`** is empty and not **`show-message`**; optional **select-all** strip: **`select-all-visible`**, **`select-all-checked`** / **`select-all-indeterminate`**, **`select-all-label-unchecked`** / **`select-all-label-checked`**, **`select-all-changed(bool)`**; **`Flickable`** + **`ActionList`** in list chrome; **`body_spinner`** uses **`SelectPanelLoading`**; **`body_skeleton`** uses **`SkeletonBox`** + **`FilteredActionListTokens`** skeleton spacing; **`PrimerTextInput`** + **`Icons.search`**; **`LayoutTokens`** + **`PrimerColors`** + **`SelectPanelTokens`**.
